# 第二百九十二章 母女

柳絮冷漠着一张脸，跟石雕一般听完吴彩霞的自述和诚恳的道谢，眼中的情绪却无半点波动。
　　只轻轻地“嗯！”了一声，就完了。
　　她这副样子，还真不是柳絮刻意装逼或者真的冷酷无情。
　　原因还是以前长时间一个人不说话不交流，柳絮自己都忘了跟人交谈脸上还需要维持表情。
　　当然跟多多一起柳絮是发自真心的，并不存在什么刻意伪装。
　　人们之间说话交流，多数是根据对方的表情变化以及肢体动作，做出相应的调整与接洽，这才是交流的真正意义。
　　吴彩霞想拉近一些两人之间的关系，自己先起了话头，可是柳絮不接话，就只是嗯了一声，这叫人还怎么往下接？
　　于是吴彩霞只能接着自己的话题往下硬聊：
　　“家里实在没吃的了，村尾的柳虎说是只要把我嫁过去，就给我们家五斤肉。
　　可是现在都什么时候了？哪里来的肉？可是我爸却…………”
　　吴彩霞十分悲痛地将自己悲惨的身世深情并茂对着柳絮一顿倾吐。
　　柳絮依旧不发表评论，自顾自的做着自己的事儿，偶尔抬头看一眼哭的梨花带雨的小姑娘，以示她有在听。
　　事情的经过大概就是，吴彩霞的爸爸，为了五斤不知道是什么的“肉”？要把还未成年的女儿嫁给一个人渣。
　　而这人渣自五年前起，已经娶了五个媳妇了，几乎是一年一个。而每个媳妇儿都活不过一年，最后都以死于非命结束，如果吴彩霞嫁过去，无异于是送她去死。
　　可是她的亲爸就是这么的奇葩，不知出于什么原因？还是答应了这门亲事。
　　就在成婚的前一夜，吴彩霞的亲妈不愿女儿去跳火坑，于是连夜带着吴彩霞跑了出来。
　　可是她们原本就已经饿了很久，才赶了不到两个小时的路，饥渴交加再加上体力透支，两人便昏死在了山沟里。
　　因此才惊动了多多。
　　吴彩霞的这段血泪史，被她那黄莺般的嗓子娓娓道来，简直是荡气回肠，催人泪下，十分的感人肺腑。
　　可是柳絮感觉这女娃表错了情。
　　要是此刻听故事的人换作是一个侠肝义胆的英雄好汉，怎么也能激起一些同情心或者怜悯与保护欲？
　　可问题是大家都是女人，同性相斥不是没有道理的。
　　柳絮对她们家的不管是父女，母女，还是夫妻之间的爱恨情仇悲惨事迹并不感兴趣。
　　你就算是讲述的再悲惨，柳絮也无法切身感受她的痛，只能是途增烦恼。
　　尤其是吴彩霞那个妈，柳絮就奇了怪了，她可是救了她们的命呀！？
　　可看她那是什么眼神儿，真的是既忌惮又渴望还不敢。
　　柳絮真的觉得她都要被自己的一百个心眼子给绕死了。
　　一直都是吴彩霞自己一个人在那儿尬聊，柳絮作为一个听众全程基本闭口不言。
　　吴彩霞说到最后，感觉自己的喉咙都要冒烟了，柳絮适时的递上一碗水。
　　吴彩霞接过水碗刚抿了一口，忽然意识到这水是柳絮凭空变出来的，顿时惊的语无伦次起来：
　　“这，这，水，你，水——”结巴到后来，直接卡壳了，张大的嘴巴甚至都能塞进去一个鸡蛋。
　　吴彩霞的母亲时刻都注意着柳絮的一举一动，看到柳絮的一番神奇操作，同时亦惊得呆住了。
　　那饿的早已脱了相深陷的眼窝中，两个大眼珠子似要脱框飞出来。
　　由此更加坚定了她心中的某些想法。
　　刚拿出水来柳絮自己也懵逼了，嘴角不由抽了抽，但是脸上麻木的肌肉依旧看着淡漠无华。
　　失误，绝对是失误。
　　一个人随心所欲惯了，都忘了山洞里还有两个人。
　　既然已经覆水难收，自然要做出补救。
　　柳絮当即又凝出一根指头粗的尖刺，尖刺通体闪着电流，嗖一下钉在了吴彩霞母亲脑袋后面的洞壁之上，女人脸都吓白了。
　　柳絮缓步上前，悠悠的看了一眼女人，当着她的面手臂轻轻一挥便收走了尖刺。
　　这是威慑，希望女人脑子清醒一点儿，千万别做什么傻事。
　　柳絮收回尖刺，返身走到洞口在椅子上坐定，回头交代了一句：
　　“好生休息，早些恢复体力，太阳落山，你们就可以离开了。”柳絮已然不耐烦应付两人，当即下了逐客令。
　　才刚喘口气，还以为这个仙人是什么良善之辈，怎么也该留她们住个两三天？
　　谁成想这么快就要赶她们走。
　　听到此话，二人脸色均是大变。
　　吴彩霞的母亲刚要拉下脸来哀求，没想到吴彩霞比她妈更着急。
　　一步上前抢在她妈之前，拽着柳絮的袖子开始哭求：
　　“神仙姐姐，你行行好，我知道你是天下最好，最善良的神仙了，求求你让我们留下吧。
　　不然你现在让我们离开，你叫我们去哪儿？这不是重新叫我们去死嘛。
　　你放心，我们吃的很少的，你只要给一口就行，我们不挑的。
　　求求你，让我们留下吧！”
　　柳絮后面的话还没说完，吴彩霞一番连珠炮后，吴彩霞的妈又扑上来挫着手跪求：
　　“大仙，大仙我知道你是厉害人，您既然救了我们，就送佛送到西，再救我们一回吧。
　　俗话说，救人一命胜造七级浮屠。您救了我们的命，也能帮助你修行不是。
　　求求您了，大仙，求求您了。”
　　母女二人，小的抱着她的胳膊哭，老的一边嚎一边磕头，柳絮顿时脑瓜子嗡嗡直响。
　　被这母女二人扰的烦不胜烦。
　　吆喝，这还要挟上了。
　　果然好人难当，末世里的好人更是不好当。
　　柳絮已经后悔了，她就不该一时心软，将两个麻烦带回来。
　　真是请神容易送神难，这是要赖上她了吗？
　　柳絮气的脸都黑了，冷漠的脸上瞬息之间挂上了摄人的寒霜。
　　“都给我住嘴！”柳絮暴怒之下一掌拍在洞口的土墙上，土墙立时哗啦啦掉下来一大块儿。
　　柳絮将她们带回来，一直都是好生相待，使得二人心生侥幸，以为柳絮是那种温柔好拿捏的性子。
　　柳絮这一声暴喝可把二人吓得不轻，顿时收住声音呆呆的望着柳絮。
