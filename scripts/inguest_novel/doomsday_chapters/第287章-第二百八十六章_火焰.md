# 第二百八十六章 火焰

柳絮瞬息心念即至意识海中心，只见原本万年都不见得会抖动一下的蓝色火焰，这会儿就跟吃了兴奋剂一样，火苗蹿的老高。
　　而且火苗的颜色似乎发生了微小的变化。
　　柳絮第一次看见它时是淡淡地清澈的水蓝色，她修炼了千年，命源的颜色也从未发生过一丝变化。
　　可这会儿看着颜色好像深了些，甚至看久了火苗中心好像其中还泛着紫。
　　柳絮以为是自己一时眼花了，闭了闭眼睛再看，还真是，不仅火焰变成了深蓝色，而且火焰的体积也增大了好几圈。
　　火焰最初的体积只有鸡蛋大小，吸收了这个世界的“柳絮”的部分命源，火焰也只不过加了一个镶边而已。
　　可此刻何止是大了一圈，柳絮目测这玩意儿都快赶上鹅蛋大了。
　　柳絮仔细回想当时的场景，她没做什么呀？当时天地也并没有发生其他意向。
　　她就是感慨生命不易，对生命进行了重新定义。
　　难道这就是传说中的顿悟！？
　　柳絮已经不是修炼小白了，通过不断的修炼摸索，即便是在修炼中有很多她不懂的，一旦时机到了，并不需要别人特别去教，也会自然而然明白其中的规则。
　　这便是一通则百通。
　　柳絮盯着命源观察了许久，终究是没看出来，变大了究竟有什么用。
　　那个神经兮兮的女人，随时都会找来，柳絮迫切想要学会一门保命的手段来自卫。
　　之前在陋室之中，没有了生命危险，整天又无忧无虑，柳絮只一门心思研究那些有的没的。
　　却从未将心思放在提高武力值上，眼下人家都要杀上门来了，她却只有抓瞎一途，半点儿自保的能力都没有。
　　现在想来简直就是在虚度光阴。
　　可是如今再后悔又有什么用，时间又不能倒流。
　　正所谓有压力，才有动力。
　　自己的生命受到了威胁，柳絮被逼着开动大脑，努力想着自己究竟有什么能拿的出手的。
　　思来想去，她吸收了那么多年的能量，柳絮不信，难道就只是为了成就这么一具人类躯体？
　　那其他的能量呢？￣へ￣
　　总不至于是被她给吃了吧！？
　　柳絮望着自己的小腹发呆，意识缓慢的沉入了丹田。
　　意识观测下丹田中的气海漩涡，正在不紧不慢的匀速旋转着，偶尔有小颗粒的能量分子被卷入其中。
　　柳絮推测那可能是空气中散落下的其他生灵的灵魂之力，或者宇宙中穿过大气层漏进来的暗能量。
　　宇宙中的规则没有什么是完美的，任何事情有利既有弊。
　　地球被大气层包裹着，被地球磁场保护着，但也因此隔绝了宇宙中的暗能量侵入。
　　没有灵气滋养，也因此地球上的生命，永远只能是普通的生命，亿万年的时光，也未能使地球孕育出什么惊世骇俗的存在？
　　或许恐龙便是一个例子，还没等长起来，只需稍稍拨动一颗陨石的轨迹，便能让其消失在萌芽状态，一切进化步骤皆归于零。
　　柳絮感觉自己这个丹田气海就像个无底洞，只进不出。
　　柳絮伸出左手，想像在陋室中那样努力凝聚力量。
　　当时只是能量组成的灵魂，能量的运行轨迹基本看的一清二楚。如今变成血肉之躯，柳絮一下子忽然不知道该如何引导？
　　柳絮从未学习过正统的修炼法门，经脉走势她基本上是一窍不通。
　　眼下性命堪忧，也只能死马当活马医了。
　　柳絮凭着往日的修炼经验，试着从丹田气海中抽出一丝能量，寻着平日里的修炼路径，用蛮力引导至手心。
　　得亏当初任意门给她再造的身体各方面的指数均已调至最佳，不然照柳絮如此折腾，怕是早已经脉俱断，变成一个废人，哪还有接下来她什么事儿？
　　所以运气这种事儿可以说是世间最难琢磨的法则。
　　柳絮在想，既然我可以将能量化作圆珠，那为什么不可以将其聚显为其他形状？
　　比如刀，剑，冰锥，或者其他什么杀伤力更大的武器？
　　一旦近身可以来个出其不意，万一不成也可以远距离进行攻击。
　　柳絮以前也曾看过几部仙侠剧，感觉像是上辈子的事儿了。
　　那些上仙啊，上神啊！就算是土地公公手中的拐杖，也是能够随叫随到，伸手便有的。
　　以前看电视只图个乐子，当然知道那是特效搞的。
　　但是细想其中的原理，可不就是能量聚显为实物的最佳范例吗？
　　柳絮并非第一次制作珠子，但却是第一次在现实时空中做这件事儿，不免有些紧张。
　　留给她的时间不多了，柳絮并没有在陋室中没完没了的试错时间，最好能够一次成功。
　　柳絮极力平复好心情，精神高度集中，为了中途不出任何纰漏，她还下意识的再次用意识扫描了一遍周围的环境。
　　确定周边的确无人打扰，柳絮才又再次投入精神测试。
　　还好形成珠子的过程，她已进行了千百年，修炼的每一个步骤早已烂熟于心。
　　很快柳絮便在掌心凝聚了一颗中心不断带着闪电的玻璃球。
　　第一步测试很成功，柳絮很高兴，将玻璃球放在一边，开始进行第二步。
　　柳絮集中意念在掌心凝聚了三个玻璃球的能量，她努力控制着能量不断拉长变形，希望将他错成一把小剑。
　　看来并不是把把都能够成功，这次实验并不是很顺利。
　　当能量被拉扯到一定程度，似是哪部分的力道施加的不均匀，啪一下崩断了。
　　能量顿时星星点点溃散在空中，胸前的多多赶忙飞起来，一口气将所有能量吸进了自己体内。
　　“哈哈！”柳絮看的也是好笑，不禁笑出了声：“你呀！一点面包渣儿你都不放过。”
　　每次和主人互动，乌木牌都能感受到主人发自真心的宠爱，柳絮也能感受到乌木牌欢快兴奋的心情。
　　两个不同的物种在各自的精神世界里互相慰藉，何尝不是一种幸福。
　　柳絮逗了一会儿多多，失败所带来的阴云消散了不少，精神也放松了。
　　再次开始尝试，便没那么紧张了。
