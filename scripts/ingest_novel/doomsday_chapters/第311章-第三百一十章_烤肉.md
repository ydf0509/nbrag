# 第三百一十章 烤肉

柳絮起身看了一眼秦明那瘦了吧唧的身板儿，和他那无精打采的样子。
　　心中哀叹：看来这事儿还得她一个女的来干，她可真是天生的劳碌命啊！
　　于是在秦明呆滞的目光注视下，柳絮一手扛一个踩着小碎步噔噔噔出了地下室。
　　到了室外，看日头现在大约是下午三点左右，空气几乎都要被烤化了。
　　柳絮眯着眼睛看了一眼天上的太阳，纠结着是要在院子里找块地挖个坑埋了，还是找点儿燃料办个火化仪式。
　　要处理的尸体可是秦明的至亲之人，柳絮觉得这种事还是秦明自己决定的好。
　　“你想怎么处理？……我建议你如果有燃料的话，最好火化。”柳絮转头问秦明自己的意见，顺便给出自己的建议。
　　这个时节不管埋在哪里，尸体腐烂的味道，总归最后都会招来不必要的麻烦。
　　到时候再被有心之人挖出来，秦明的父母恐怕死都得不到安宁。
　　但如果选择火化，一把火全部烧干净了一了百了，也省了记挂。
　　秦明默默的进到别墅里面，出来时手里已经提了一桶五升装的菜籽油。
　　看到那桶菜籽油，柳絮脸上的肌肉忍不住抽了一下。
　　好吧！有总比没有好。~_~
　　体力活儿柳絮已经帮秦明干了，剩下的事就留给他自己吧。
　　秦明一边抽泣一边将菜籽油均匀的泼洒在尸体上，注目良久，却一直不肯动手。
　　就这么一会儿时间，身上的衣服已经被汗水湿透了。
　　柳絮见他眼泪在眼圈儿里打转，又要决堤的姿势。
　　也不等秦明发话，主动帮他划了一根火柴弹到了尸体上。
　　火柴碰到尸体上的热油，瞬间轰的一下火焰蹿起来老高，炽热的高温烤的两人眉毛都卷了起来。
　　柳絮连忙拉着秦明退到了室内。
　　秦明眼中要掉不掉的泪水终于再次爬满了脸颊，悲从中来大声哭道：“爸，妈！儿子不孝！……”
　　柳絮依旧木着一张脸站在一旁一言不发，任由秦明发泄心中的悲愤。
　　在她看来，生命的死亡即使脱离肉体，无外乎从一种形式转化成为另外一种形式。
　　即便是充作宇宙能量，生命也从来没有消失过。
　　普通的人类非死不得见，又怎能参透其中的奥义。
　　菜籽儿油被点燃后，一股油煎烤肉的香味弥漫在整个别墅区周围，顿时香味飘出去老远。
　　刚才秦明提那桶油过来，她就感觉有些不妥，此刻再说什么都已经迟了。
　　柳絮担忧地皱眉看向身侧的秦明，而秦明却仍然沉浸在悲痛之中无法自拔，浑然不觉危险即将逼近。
　　别墅周围的电网看似很安全，但只要有人稍微动一下脑筋，带上工具过来将电网剪断，再群起而攻之。
　　以秦明的战武渣体质，恐怕连一秒钟都顶不住，别墅就要沦陷了。
　　柳絮今夜就要出发赶往武兴市，她半点儿都不想参合别人的闲事。
　　可看着秦明那若不经风楚楚可怜的样子，柳絮终也是狠不下心来就此离去。
　　于是抬起下巴朝着尸体努了努嘴，点醒秦明。
　　无奈之下翻了个大白眼儿示意他往那儿看：“你准备接下来怎么办？”
　　秦明起初还没明白柳絮什么意思，不过鼻尖萦绕着的那股香气四溢的烤肉味儿却是挥之不去。
　　秦明顿时睁大眼睛，吓得身体一个哆嗦，将他的思绪从悲伤的深渊中拉了回来。
　　秦明也没想到，自家烧尸体也能给自己招来祸端。
　　话已递到，柳絮便不再多言，丢下已经六神无主的秦明，自顾自地进屋去了。
　　外面的空气实在太热了，继续陪这个傻得烤地瓜，简直就是自己找虐，还是屋里的空调房更舒服些。
　　离太阳落山还有好几个小时，柳絮觉得这段时间足够他想出应对之法了，便不再管他。
　　末世里人人自危，即便是七八岁的小孩儿，五年的艰苦环境磨练下警戒意识也应该已经调至最高。
　　而秦明一直被保护的象牙塔里，又衣食无忧，导致他的生存意识极差。
　　末世里的生存法则是：弱肉强食，适者生存。
　　现在与他相依为命的父母也去世了，秦明也该成长起来了。
　　柳絮来到地下室，接收到中央空调吹来的凉风，给自己倒了一杯空间出品的清凉泉水，一边自顾自惬意的喝着，一边计划着自己之后的行程。
　　约摸过了大约半个小时，秦明忽然风风火火的跑到柳絮面前，紧张的问道：
　　“姐姐，你接下来是不是要去武兴市？”
　　“你问这个干嘛？”柳絮不答反问，她有种不好的感觉。
　　果然，就听秦明下一刻便弱弱的道：“那个……姐姐你能不能带我一起走？我一个人住在这里好害怕。”
　　柳絮挑眉像看傻子一样诧异地看着秦明，心里吐槽：你害怕管我屁事，老娘又不是保姆，还能保你一世不成。
　　再看他那张堪比明星的俊脸，柳絮似乎悟到了什么不得了的事儿：
　　难道他以为单凭他这张小白脸儿就能打动我！？→_→
　　柳絮再次挑眉冷冷拒绝道：“去不去武兴市是我自己的事，带上你反倒会拖慢我的脚步。”
　　其实柳絮已经说的很委婉了，没说他是个累赘已经很给他面子了。
　　听到柳絮的答话，秦明显得怅然若失，但是还是依旧不死心道：
　　“姐姐是这样子的，我还有一个叔叔在武兴市，现在家里就剩我一个了，我想去找他。
　　但是凭我一个人，根本就到不了那里，我希望能跟你同路。”
　　柳絮在想自己一个人上路，以她的速度一个晚上就能到目的地。
　　可要是带上秦明，又没有代步工具，按照普通人的脚程，到达武兴市还不得磨上好几天，实在没那个必要。
　　柳絮正想要放句狠话拒绝他的无理要求，秦明突然走到一个房间门口，用钥匙打开门叫柳絮过去。
　　柳絮不知道他想要搞什么？但还是跟着走到门口往里看。
　　“姐姐你看，如果你能带上我一起上路，这间屋子里所有的食物就都是你的，这算作是我的保护费如何？”
　　有了雄厚的物资做后盾，秦明说话的底气都足了很多。
